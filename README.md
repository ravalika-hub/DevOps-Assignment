# DevOps Assignment Monorepo

##Structure
- 'backend/':FastAPI app
- 'frontend/':Next.js app

##Branching Strategy
- 'main': Production
- 'develop': Active development
- 'feature/*': Feature branches
- 'hotfix/*': Hotfixes

##Setup Instructions
1. Clone repo
2. Install dependencies in 'backend/' and 'frontend/'
3. Run using Docker or 'yarn dev' / 'uvicorn'
